Our compass

Travel Guide blends editorial storytelling with hands-on tools — because the best trips start with curiosity and a little planning magic.

Why this project

This site is built for Web Technologies 1: semantic HTML, responsive CSS, and modular JavaScript. Arrays drive our destination cards; event listeners power the contact form; localStorage remembers your theme between visits.

On the Flag Quiz page we fetch live country data from a public API so every round feels fresh — with graceful fallbacks if you are offline.
